# My Hero Academia Recommendation System

A Python-based recommendation system that suggests *My Hero Academia*
characters to users based on their ratings and character attributes.\
The project demonstrates **collaborative filtering**, **content-based
filtering**, and a **hybrid recommendation approach**, along with data
visualizations and a simple web interface.

------------------------------------------------------------------------

## Project Overview

This project showcases how recommendation systems work using explicit
user ratings and content features.\
It analyzes user preferences and character traits to generate
personalized recommendations and visual insights.

The goal is to demonstrate **clean software design**, **modular
architecture**, and **practical machine learning concepts**.

------------------------------------------------------------------------

## Skills Demonstrated

-   Python software design and modular architecture
-   Recommendation systems (collaborative, content-based, hybrid)
-   Data processing with pandas
-   Data visualization with matplotlib and seaborn
-   Unit testing with unittest
-   Streamlit web applications
